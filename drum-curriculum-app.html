LOGIC PRO WORKFLOW

Every song is tracked in Logic Pro. Drums go down first via Superior Drummer, then layer instruments one at a time.

Phase 1: Drum Tracking

Record drums via MIDI into Superior Drummer. This is the foundation for every song module.

  • Create a new Logic project per song (use the song slug as the project name)
  • Set up a Software Instrument track routed to Superior Drummer
  • Set the project tempo to the song's starting BPM from the practice ladder
  • Record the core groove as a 2-4 bar MIDI region, then loop it
  • Apply the song's quantize strategy (see each song's Logic notes)
  • Apply humanize settings to preserve feel after quantization
  • Build the arrangement by duplicating and varying regions per song section
  • Export a rough drum mix as a reference bounce

Phase 2: Layer Guitars

Once the drums feel locked, add guitar parts. Each song specifies roles, chord ideas, and effect notes.

  • Add an Audio track for guitar (or Software Instrument for amp sims)
  • Reference the song's guitar role — rhythm, texture, lead, etc.
  • Record to the drum arrangement — let the drums guide your phrasing
  • Apply effects per the song's guitar effect notes

Phase 3: Layer Bass

Bass locks to the kick drum. Most songs specify how bass should relate to the drum pattern.

  • Add a Bass audio or instrument track
  • Follow the song's bass pattern idea — most lock directly to kick
  • Verify kick/bass alignment in the piano roll or waveform view
  • Apply the song's bass effect notes

Phase 4: Layer Moog / Synth

Synth parts fill atmosphere and often lead section changes. Add these last.

  • Add a Software Instrument track (ES2, Alchemy, or external synth)
  • Follow the song's Moog motif idea and modulation notes
  • Automate parameters (filter, volume) per the song's notes
  • Check that synth parts don't compete with drums for space

Superior Drummer Tips (Global)

  • Map closed hats and open hats to separate MIDI notes for clear articulation
  • Keep snare velocity zones distinct: 35-45 for ghosts, 70-80 for light notes, 105-120 for backbeats
  • Start with tight modern rock kits, lower room mics for clarity at speed
  • Use the MIDI mapping tips in each song's Logic tab for kit-specific guidance
